🔧 One-time setup: Add your OAuth Client ID

  1. Go to console.cloud.google.com/apis/credentials
  2. Click + Create Credentials → OAuth client ID
  3. Application type: Web application · Name: teamtaika Analytics
  4. Authorized JavaScript origins: add https://teamtaika.com and http://localhost
  5. Copy the Client ID (ends in .apps.googleusercontent.com)
  6. In this file (pages/admin/analytics.html), replace 253845963985-5j7e8dngcvva5fqc6s94g2dhepjdibmb.apps.googleusercontent.com with it
  7. Also enable: Google Analytics Data API and Search Console API under APIs & Services → Library
